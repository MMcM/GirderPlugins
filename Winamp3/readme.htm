Girder plug-in for Winamp 3.x

Version 1.0.  MMcM 08/02

Use

This plug-in interfaces to Winamp version 3.

It provides bidirectional communication between Winamp and Girder, sending player state changes as Girder events and allowing Girder commands to in turn affect / query the state.

Files

These are the files contained in this archive and what purpose they serve.
  • readme.htm
    This file.

  • Winamp3.dll
    The Girder plug-in proper.

    This file must be placed in the Plugins folder beneath where Girder is installed.

  • girder.wac
    The Winamp component for Girder.

    This file must be placed in the wacs folder beneath where Winamp is installed.

  • preferences.xml
    The "skin" for the Girder preferences section.

    This file must be placed in a wacs\xml\girder folder beneath where Winamp is installed. It may be simplest to copy both girder.wac and the whole xml subfolder from the directory where the archive was unzipped into the wacs folder under Winamp3 using the Explorer.
