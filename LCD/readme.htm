Girder plug-in for LCD/VFD displays

Version 1.10.  MMcM 05/02

Prerequisites

Before you can use this plug-in, you must install and configure LCDisplay. You may also want to test that it works by itself before proceeding. Any hardware that works with that driver ought to work with this plug-in.

Use

This plug-in displays bits of text on chosen area of the display. The area is specified by the starting row and column and the number of characters. The text can come from several places:
  • A fixed string, possibly expanding registers ([regn]) or text registers ([tregn]).
  • The contents of a string register, as set by some other plug-in.
  • The current date and time, formatted using a strftime control string.
Several special actions that are also possible:
  • Clearing the display.
  • Closing the display so that it can be used by other applications.
  • The contents of a payload string.
  • The contents of a payload string, interpreted as a filename. Any directory and file type portion of the name are stripped before display.

The most interesting use is with the DVDSpy Hardware plug-in, which generates events when the state of a software DVD player changes. These events are accompanied by text in string register #1. See the readme file for that plug-in for more details. A sample Girder file is included that displays some information in a format suitable for 4x20 displays.
