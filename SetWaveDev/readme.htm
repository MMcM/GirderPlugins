Girder plug-in for changing sound playback / record device

Version 1.3.  MMcM 08/02

Use

This plug-in changes which sound card is used by default for audio playback or recording. This is useful for machines with one sound card for games (usually from the SoundBlaster family) and another for hifi (usually a 24bit / 96KHz card).
