Girder plug-in for decoding ID3 tags

Version 1.0.  MMcM 08/02

Use

This plug-in extracts information from an ID3 tag file, such as an MP3 file.

Files

These are the files contained in this archive and what purpose they serve.
  • readme.htm
    This file.

  • ID3.dll
    The Girder plug-in proper.

    This file must be placed in the Plugins folder beneath where Girder is installed.

  • Test.gml
    A test script that takes some tags from the current Winamp track and displays them in a PopUp OSD.
